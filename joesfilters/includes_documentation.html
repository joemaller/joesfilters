whatType()

This returns a 0, 1 or 2 depending on the type of effect it's placed into. The function works by using the return-error handling behavior to check for the existence of src1 and src2. If both exist, it's a transition, it only Src1 exists, it's a filter. If neither exist, it's a generator.
